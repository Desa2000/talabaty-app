# Talabaty App

A food and product delivery platform for Sudan connecting customers with restaurants, supermarkets, and local shops in one application.

This project contains 4 user roles in a single Flutter application:
1. Customer
2. Courier
3. Merchant
4. Admin

## Getting Started

### Prerequisites
- Flutter SDK
- Android Studio or VS Code with Flutter extension

### Installation

1. Open this project folder (`talabaty_app`) in Android Studio.
2. Run the following command in the terminal to get all dependencies:
   ```bash
   flutter pub get
   ```
3. Connect an Android emulator or a physical device.
4. Run the app:
   ```bash
   flutter run
   ```
